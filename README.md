# Генератор случайных чисел

Простая GUI-программа на Python для генерации случайного числа в заданном пользователем диапазоне.

## Возможности

- Генерация случайного целого числа в указанном диапазоне
- Проверка корректности введенных данных
- Обработка ошибок (нечисловые значения, пустые поля, min > max)
- Интуитивно понятный интерфейс

## Требования

- Python 3.x
- Библиотека `tkinter` (входит в стандартную поставку Python)

## Установка и запуск

1. Скачайте файл `random_generator.py`

2. Запустите программу через терминал/командную строку:
   ```bash
   python random_generator.py
